# Faber-Aeternus 🦀

The eternal-craftsman BYOA (Bring Your Own Agent) coding harness.

## Features
- **Zero bloat**: Assembled natively via Rust. 
- **TUI**: Powered by `iocraft` with a sleek Catppuccin Macchiato design.
- **Security-first**: Bubblewrap file sandboxing mapped securely directly below the process layer.
- **Provider agnostic**: Remote endpoints or local running via `candle-core`.

## Getting Started

```bash
cargo check --all-features --tests
cargo run -- --provider grok
```

## Philosophy  
Faber-aeternus acts as your hyper-capable eternal companion. Written out of pure performance principles.
